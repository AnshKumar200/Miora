# Miora
Note taking webapp

## Getting Started
Follow these instructions to set up and run the webapp locally.

### Prerequisites
-

### 1. Clone the Repository
```bash
git https://github.com/AnshKumar200/Miora.git
cd Miora
```

## 2. Install Dependencies
Install all the required packages for the project.
```bash
npm install
```

## 3. Run the webapp
```bash
npm run dev
```
